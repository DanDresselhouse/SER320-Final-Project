SER340 is the software requirements analysis class. This class covers proper elicitation of requirements from clients and how to appropriately manage the requirements. This class also covers basic HTML, CSS, and Javascript.

SER305 is the advanced computational problem solving class. This class teaches students how to solve problems creatively while also constraining the problems to specific space and time requirements. This class covers topics such as bitwise, dynamic programming, programming with tree structures, etc.
